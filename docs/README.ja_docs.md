## 📚 コース情報

このプロジェクトは、シドニー工科大学(UTS)の**41025 - Introduction to Software Development**コースの一部として開発されました。

- **コースコード**: 41025
- **コース名**: Introduction to Software Development

**IoT Bay**は、**UTS（シドニー工科大学）の学術課題 (41025 Introduction to Software Development - Assignment 2 Autumn 2025)**の一部として開発された、IoTデバイス管理とEコマースのための現代的でレスポンシブなWebアプリケーションです。**JSP**、**Java MVC**、**Maven**、**Jettyサーバー**で構築されており、包括的なデザインシステム、ダークモードサポート、レスポンシブグリッドレイアウト、エンタープライズグレードのセキュリティ機能を特徴としています。

### プロジェクト目標

- IoTデバイスの完全に機能するEコマースプラットフォームの実装
- JSP/Servlet Web開発の習熟度の実証
- MVCアーキテクチャパターンの適用
- 安全なユーザー認証と認可の実装
- 直感的で現代的なユーザーインターフェースの作成
- データベース設計とデータアクセスパターンの実証
- エンタープライズグレードのセキュリティ対策の実装

## 主な機能

### モダンなUI/UXデザイン
- **レスポンシブデザイン**: カスタムCSSを使用したモバイルファーストアプローチ
- **ダークモード**: ライト/ダークテーマの切り替え

### 管理者機能
- **スタッフダッシュボード**: 分析とレポート

- **製品管理**: 在庫とカタログ管理
- **データ管理**: システムデータ管理

### セキュリティ機能
- **サーバーサイド検証**: 包括的な入力検証
- **CSRF保護**: トークンベースの保護

## クイックスタート

### 前提条件

- **Java JDK 8+** インストール済み
- **Maven 3.6+** インストール済み
- 依存関係のための**インターネット接続**

### インストールとセットアップ

1. **リポジトリのクローン**
   ```bash
   git clone <repository-url>
   cd IoTBay
   ```

2. **プロジェクトのビルド**
   ```bash
   mvn clean install
   ```

3. **アプリケーションの実行**
   ```bash
   mvn jetty:run
   ```

4. **アプリケーションへのアクセス**
   ```
   http://localhost:8080/
   ```

## プロジェクト構造

```
IoTBay/
├── src/main/java/
│   ├── controller/           # サーブレット (MVCコントローラー)
│   ├── dao/                  # データアクセスオブジェクト
│   │   ├── stub/            # テスト用スタブ実装
│   │   └── impl/            # データベース実装
│   ├── model/               # JavaBeans (User, Product, Order)
│   ├── service/             # ビジネスロジック層
│   ├── utils/               # ユーティリティクラス (セキュリティ、検証、エラー処理)
│   └── config/              # 設定 (DIContainer)
├── src/main/webapp/
│   ├── components/          # 再利用可能なJSPコンポーネント
│   │   ├── header.jsp       # ナビゲーションヘッダー
│   │   ├── footer.jsp       # サイトフッター
│   │   └── layout/          # レイアウトテンプレート
│   ├── css/
│   │   └── modern-theme.css # デザインシステムを含むメインスタイルシート
│   ├── js/
│   │   └── main.js          # JavaScript機能
│   ├── images/              # 静的アセット
│   ├── WEB-INF/
│   │   ├── web.xml          # デプロイ記述子
│   │   └── views/           # 保護されたJSPページ
│   └── *.jsp                # JSPページ
├── docs/                    # プロジェクトドキュメント
│   ├── architecture/        # システム設計と決定
│   ├── requirements/        # 機能と仕様
│   ├── testing/             # テスト戦略と監査
│   ├── development/         # 開発者ガイド
│   ├── plans/               # 実装計画
│   ├── reports/             # ステータスレポート
│   └── archive/             # 履歴データ
└── pom.xml                  # Maven設定
```

## 設定

### ポート設定

デフォルトポートは**8080**です。変更するには、`pom.xml`を修正してください:

```xml
<plugin>
    <groupId>org.eclipse.jetty</groupId>
    <artifactId>jetty-maven-plugin</artifactId>
    <configuration>
        <httpConnector>
            <port>8090</port>
        </httpConnector>
    </configuration>
</plugin>
```

### データベース設定

現在、開発のために**DAO Stubs**を使用しています。実際のデータベースを使用するには:

1. `src/main/resources/database.properties`を更新
2. `src/main/java/dao/impl/`にデータベースDAOクラスを実装
3. `web.xml`サーブレット設定を更新
