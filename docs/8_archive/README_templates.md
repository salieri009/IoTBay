# Templates

Page-level layouts without specific content.

## Components

- **page-base** - Base page template (header + footer + main)
- **two-column** - Two-column layout template
- **dashboard** - Dashboard layout template
- **auth** - Authentication page template

## Usage

```jsp
<t:page-base title="Page Title">
  <!-- Page content -->
</t:page-base>
```

## Rules

1. Templates define page structure
2. No specific content, only layout
3. Use organisms for major sections
4. Provide content areas for pages
