## Overview

The IoT Bay Design System provides a unified language for our user interface, ensuring consistency, accessibility, and efficiency.

## Core Principles

1.  **Atomic Design**: Components are built from atoms up to pages.
2.  **Accessibility First**: WCAG 2.1 AA compliance is mandatory.
3.  **Responsive**: Mobile-first approach.
