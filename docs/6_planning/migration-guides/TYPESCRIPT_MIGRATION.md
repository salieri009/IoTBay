## Overview

This guide outlines the plan to migrate legacy JavaScript code to TypeScript to improve type safety and maintainability.

## Strategy
1.  **Setup**: Configure `tsconfig.json` and install dependencies.
2.  **Incremental Migration**: Rename `.js` files to `.ts` one by one.
3.  **Strict Mode**: Enable `strict: true` gradually.
