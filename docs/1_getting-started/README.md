## ðŸ†˜ Common Questions

**Q: Where do I find the code?**  
A: Backend code is in `src/main/java`, frontend JSP files are in `src/main/webapp`.

**Q: How do I run the application?**  
A: Execute `mvn jetty:run` and visit `http://localhost:8080`.

**Q: What if I'm stuck?**  
A: Check [Debugging Guide](../4_development/DEBUGGING_GUIDE.md) or ask your team lead.

**Q: Where are the tests?**  
A: Tests are in `src/test/java`. Run them with `mvn test`.

**Q: What's the commit message format?**  
A: See [Contributing Guidelines](../4_development/CONTRIBUTING.md).
