**コース**: 41025 Information Systems Development (ISD)  
**課題**: Assignment 2 - Autumn 2025  
**機関**: University of Technology Sydney (UTS)  
**プロジェクトタイプ**: IoTデバイスEコマースWebアプリケーション

---

**IoT Bay**は、**UTS（シドニー工科大学）の学術課題 (41025 ISD Assignment 2 Autumn 2025)**の一部として開発された、IoTデバイス管理とEコマースのための現代的でレスポンシブなWebアプリケーションです。**JSP**、**Java MVC**、**Maven**、**Jettyサーバー**で構築されており、包括的なデザインシステム、ダークモードサポート、レスポンシブグリッドレイアウトを特徴としています。

### プロジェクト目標

- IoTデバイスの完全に機能するEコマースプラットフォームの実装
- JSP/Servlet Web開発の習熟度の実証
- MVCアーキテクチャパターンの適用
- 安全なユーザー認証と認可の実装
- 直感的で現代的なユーザーインターフェースの作成
- データベース設計とデータアクセスパターンの実証

## 主な機能

### モダンなUI/UXデザイン
- **レスポンシブデザイン**: カスタムCSSを使用したモバイルファーストアプローチ
- **ダークモード**: ライト/ダークテーマの切り替え
- **コンポーネントベースアーキテクチャ**: 再利用可能なJSPコンポーネント
- **グリッドレイアウト**: モダンな製品グリッドシステム
- **インタラクティブ要素**: スムーズなアニメーションとホバー効果

### Eコマース機能
- **製品カテゴリ**: 産業用、倉庫、農業、スマートホーム
- **製品閲覧**: 高度なフィルタリングと検索
- **ショッピングカート**: カートに追加機能
- **ユーザー認証**: ロールベースアクセス権限を持つログイン/登録
- **注文管理**: 注文履歴と追跡

### ユーザー管理
- **顧客アカウント**: プロフィール管理と注文履歴
- **スタッフダッシュボード**: 管理ツールと分析
- **ロールベースアクセス**: 顧客とスタッフに対する異なる権限
- **セッション管理**: 安全なユーザーセッション

### 管理者機能
- **スタッフダッシュボード**: 分析とレポート
- **ユーザー管理**: 顧客およびスタッフアカウント管理
- **製品管理**: 在庫とカタログ管理
- **データ管理**: システムデータ管理

## クイックスタート

### 前提条件

- **Java JDK 8+** インストール済み
- **Maven 3.6+** インストール済み
- 依存関係のための**インターネット接続**

### インストールとセットアップ

1. **リポジトリのクローン**
   ```bash
   git clone <repository-url>
   cd IoTBay
   ```

2. **プロジェクトのビルド**
   ```bash
   mvn clean install
   ```

3. **アプリケーションの実行**
   ```bash
   mvn jetty:run
   ```

4. **アプリケーションへのアクセス**
   ```
   http://localhost:8080/
   ```

## 設定

### ポート設定

デフォルトポートは**8080**です。変更するには、`pom.xml`を修正してください:

```xml
<plugin>
    <groupId>org.eclipse.jetty</groupId>
    <artifactId>jetty-maven-plugin</artifactId>
    <configuration>
        <httpConnector>
            <port>8090</port>
        </httpConnector>
    </configuration>
</plugin>
```

### データベース設定

現在、開発のために**DAO Stubs**を使用しています。実際のデータベースを使用するには:

1. `src/main/resources/database.properties`を更新
2. `src/main/java/dao/impl/`にデータベースDAOクラスを実装
3. `web.xml`サーブレット設定を更新
