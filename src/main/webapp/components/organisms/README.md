# Organisms

Complex UI sections composed of molecules and atoms.

## Components

- **header** - Site header with navigation
- **footer** - Site footer
- **product-grid** - Product listing with filters
- **filter-sidebar** - Product filter sidebar
- **cart-summary** - Shopping cart summary
- **order-details** - Order detail view
- **user-profile** - User profile section

## Usage

```jsp
<jsp:include page="/components/organisms/header/header.jsp" />
```

## Rules

1. Organisms combine molecules and atoms
2. Can have complex interactions
3. May include JavaScript
4. Represent major page sections
